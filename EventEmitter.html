Description

The EventEmitter class provides methods to implement the observable design pattern. It is an abstract class meant to be extended on (or mixed in) to add methods such as on(), off(), emit(), etc.

emit( event )

Description

Executes the callback functions of all Listener objects registered for a given event. The functions are passed an Event object as the first parameter and the content of the listener's data property (if any) as a second parameter.

If suspended is true no callback functions will be executed.
